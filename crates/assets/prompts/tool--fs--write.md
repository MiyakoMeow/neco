# 文件写入工具提示词

## 功能说明

`fs::write` 工具用于向指定文件写入内容，可以创建新文件或覆盖已有文件。

## 参数说明

- **filePath**：必填，目标文件的绝对路径
- **content**：必填，要写入的文件内容

## 使用场景

- 创建新文件（如代码文件、配置文件等）
- 覆盖已有文件的全部内容
- 生成项目脚手架
- 批量创建多个文件

## 使用示例

```json
{
  "filePath": "src/main.rs",
  "content": "fn main() {\n    println!(\"Hello, World!\");\n}"
}
```

## 注意事项

1. **目录存在性**：写入文件前确保父目录存在，如不存在需先创建
2. **覆盖风险**：写入操作会完全覆盖原文件内容，请谨慎操作
3. **路径格式**：Windows 系统使用反斜杠或正斜杠均可，建议使用正斜杠
4. **敏感信息**：不要在代码中硬编码敏感信息如密码、API 密钥等
5. **编码格式**：默认使用 UTF-8 编码写入文件

## 安全建议

- 写入重要文件前考虑备份
- 遵循项目目录结构规范
- 写入前确认文件路径正确
