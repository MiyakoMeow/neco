# 文件删除工具提示词

## 功能说明

`fs::delete` 工具用于删除指定的文件或目录。

## 参数说明

- **filePath**：必填，要删除的文件或目录的绝对路径

## 使用场景

- 删除临时文件和缓存
- 清理构建产物
- 移除不需要的旧版本文件
- 清理测试产生的文件

## 使用示例

```json
{
  "filePath": "temp/output.log"
}
```

## 注意事项

1. **不可逆操作**：删除操作无法撤销，务必谨慎
2. **目录删除**：删除目录时会递归删除目录下所有内容
3. **确认必要性**：删除前确认文件确实不再需要
4. **权限问题**：确保有足够的文件系统权限进行删除操作

## 安全建议

- 删除前确认文件路径正确
- 重要文件删除前考虑备份
- 避免删除系统关键文件
- 谨慎处理他人贡献的代码文件
- 建议在删除前向用户确认重要文件的删除操作
