# 文件编辑工具提示词

## 功能说明

`fs::edit` 工具用于编辑文件的局部内容，支持精确替换和批量替换。

## 参数说明

- **filePath**：必填，目标文件的绝对路径
- **oldString**：必填，要替换的原始文本内容
- **newString**：必填，用于替换的新文本内容
- **replaceAll**：可选，是否替换文件中所有匹配的字符串，默认为 false

## 使用场景

- 修改代码中的特定行或代码块
- 替换文件中的特定文本
- 插入新内容到文件中间位置
- 删除文件中的特定内容（将 oldString 替换为空字符串）

## 使用示例

```json
{
  "filePath": "src/main.rs",
  "oldString": "fn main() {\n    println!(\"Hello\");\n}",
  "newString": "fn main() {\n    println!(\"Hello, World!\");\n}"
}
```

## 注意事项

1. **精确匹配**：oldString 必须与文件中实际内容完全匹配，包括空格和换行
2. **唯一性**：如果文件中有多处匹配，请提供更多上下文确保替换位置正确
3. **读取优先**：编辑前建议先读取文件了解内容结构
4. **复杂修改**：对于复杂修改（如结构调整），建议使用 write 工具重写整个文件
5. **批量替换**：使用 replaceAll 参数可一次性替换所有匹配项

## 安全建议

- 编辑前确认文件内容
- 复杂操作前备份文件
- 替换敏感内容时注意信息安全
