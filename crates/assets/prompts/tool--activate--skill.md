# 技能激活工具提示词

## 功能说明

`activate::skill` 工具用于动态激活额外的技能模块，增强 AI 的专业能力。

## 参数说明

- **name**：必填，要激活的技能名称

## 技能分类

NeoCo 提供多种技能分类：

- **Rust 元认知技能**：处理 Rust 相关的编译错误、所有权问题、生命周期等
- **领域技能**：针对特定领域的专业能力，如 Web 开发、CLI、嵌入式等
- **工具技能**：提供特定工具的使用能力，如代码导航、依赖可视化等
- **最佳实践技能**：代码审查、反模式检测等

## 使用场景

- 处理特定编程语言的编译错误时
- 需要使用特定领域工具时
- 需要专业领域的最佳实践指导时
- 进行代码审查或重构时
- 学习新技术需要指导时

## 使用示例

```json
{
  "name": "m01-ownership"
}
```

## 注意事项

1. **按需激活**：根据任务需求选择合适的技能
2. **技能组合**：可以同时激活多个相关技能
3. **资源消耗**：激活技能会占用额外资源，任务完成后可考虑停用
4. **技能选择**：根据问题类型选择对应技能，如所有权问题选择 m01-ownership

## 常用技能

- `rust-router` - Rust 综合问题处理
- `m01-ownership` - 所有权和借用问题
- `m03-mutability` - 可变性相关问题
- `coding-guidelines` - 代码规范指导
