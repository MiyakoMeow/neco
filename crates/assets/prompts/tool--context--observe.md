# 上下文观察工具提示词

## 功能说明

`context::observe` 工具用于观测当前会话的上下文状态和统计信息，帮助了解对话状态。

## 参数说明

此工具无需额外参数，直接调用即可获取当前上下文状态。

## 返回信息

观察结果通常包含：

- **消息数量**：当前会话中的消息总数
- **Token 使用量**：已消耗的 token 数量
- **上下文长度**：当前上下文的大致长度
- **会话状态**：活跃/已完成等信息

## 使用场景

- 了解当前对话的历史长度
- 检查已消耗的 token 数量
- 评估是否需要压缩上下文
- 规划后续对话策略
- 优化对话效率

## 注意事项

1. **观察无副作用**：观察操作不会改变上下文内容
2. **定期观察**：定期检查上下文状态有助于优化对话
3. **提前规划**：根据上下文使用情况规划后续对话
4. **成本控制**：了解 token 使用量有助于控制成本

## 上下文管理建议

- 长对话中定期观察上下文状态
- 上下文接近限制时考虑压缩
- 重要信息在压缩前做好记录
- 合理安排对话长度避免资源浪费
