# 文件列表工具提示词

## 功能说明

`fs::list` 工具用于列出指定目录中的文件和子目录。

## 参数说明

- **path**：可选，要列出的目录路径，默认为当前工作目录
- **pattern**：可选，用于过滤文件的 glob 模式

## 使用场景

- 查看项目目录结构
- 查找特定类型的文件
- 了解某个目录的内容
- 验证文件是否创建成功
- 探索项目代码组织方式

## 使用示例

```json
{
  "path": "src",
  "pattern": "*.rs"
}
```

## 注意事项

1. **路径格式**：支持相对路径和绝对路径
2. **模式匹配**：pattern 使用 glob 模式，如 `*.rs` 匹配所有 Rust 文件
3. **隐藏文件**：默认不显示以点开头的隐藏文件
4. **递归选项**：列出结果会区分文件和目录，目录通常以 `/` 结尾

## 常用模式示例

- `*.js` - 所有 JavaScript 文件
- `*.{ts,tsx}` - 所有 TypeScript 文件
- `src/**/*` - src 目录下所有文件
- `**/*.json` - 所有 JSON 文件
- `!node_modules` - 排除 node_modules 目录
