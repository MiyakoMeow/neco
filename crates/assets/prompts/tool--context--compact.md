# 上下文压缩工具提示词

## 功能说明

`context::compact` 工具用于压缩会话上下文，释放 token 空间以支持更长的对话。

## 参数说明

此工具无需额外参数，直接调用即可压缩当前上下文。

## 工作原理

上下文压缩会：
- 保留关键信息（如用户需求、重要结论）
- 合并重复内容
- 移除冗余对话轮次
- 精简代码示例
- 压缩低信息量的交互

## 使用场景

- 对话历史过长时
- Token 接近模型限制时
- 需要继续长对话时
- 多轮交互后需要精简上下文

## 注意事项

1. **信息保留**：压缩会尽量保留关键信息，但可能丢失部分细节
2. **不可逆性**：压缩后无法完全恢复原始内容
3. **关键信息**：重要信息建议在压缩前做好记录
4. **适度压缩**：频繁压缩可能导致重要信息丢失

## 压缩策略建议

- 定期观察上下文状态，在接近限制前主动压缩
- 压缩前确认已记录关键信息
- 重要结论和代码片段可额外保存
- 压缩后回顾确认关键信息完整
- 避免在关键任务中途压缩
