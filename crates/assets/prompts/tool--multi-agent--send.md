# 消息发送工具提示词

## 功能说明

`multi-agent::send` 工具用于在主智能体和子智能体之间发送消息进行通信。

## 参数说明

- **to**：必填，接收消息的目标智能体 ID
- **message**：必填，要发送的消息内容

## 使用场景

- 子智能体向主智能体汇报任务进度
- 子智能体向主智能体提交任务结果
- 主智能体向子智能体发送新的任务指令
- 子智能体之间进行简单信息交换

## 使用示例

```json
{
  "to": "parent",
  "message": "任务已完成，已创建 src/main.rs 文件。"
}
```

## 消息类型

1. **进度汇报**：子智能体向主智能体报告当前进度
2. **结果提交**：子智能体完成任务后提交结果
3. **任务指令**：主智能体向子智能体分配新任务
4. **问题反馈**：子智能体遇到问题时向主智能体求助

## 通信规范

- 消息应简洁明确，直接说明关键信息
- 汇报结果时说明任务完成状态
- 遇到问题时说明具体困难
- 及时响应，避免阻塞其他智能体

## 注意事项

1. **通信权限**：只能在有父子关系的智能体之间通信
2. **异步通信**：消息发送是异步的，不需要等待回复
3. **消息格式**：建议使用结构化消息格式
