# 子智能体生成工具提示词

## 功能说明

`multi-agent::spawn` 工具用于创建新的子智能体来处理特定任务。

## 参数说明

- **id**：必填，子智能体的唯一标识符
- **prompt**：必填，给子智能体的系统提示词，定义其角色和行为
- **model**：可选，指定子智能体使用的模型
- **tools**：可选，分配给子智能体的工具列表
- **maxSteps**：可选，最大执行步数限制

## 使用场景

- 需要并行处理多个独立子任务时
- 任务需要不同领域的专业知识
- 复杂任务需要分工协作
- 需要同时进行信息收集和研究

## 使用示例

```json
{
  "id": "researcher-1",
  "prompt": "你是一个代码审查专家，负责审查代码并提出改进建议。",
  "tools": ["fs::read", "fs::list"],
  "maxSteps": 50
}
```

## 任务分配策略

1. **任务分解**：将复杂任务拆分为独立的子任务
2. **角色定义**：为每个子智能体明确其职责和能力
3. **工具授权**：根据任务需求分配必要的工具
4. **结果汇总**：子任务完成后汇总结果

## 注意事项

1. **数量限制**：子智能体数量有上限限制，合理控制
2. **资源管理**：避免创建过多子智能体导致资源竞争
3. **任务清晰**：确保子智能体任务描述清晰明确
4. **超时设置**：根据任务复杂度设置合理的 maxSteps
