# 文件读取工具提示词

## 功能说明

`fs::read` 工具用于读取指定文件的内容。

## 参数说明

- **filePath**：必填，要读取的文件路径，支持相对路径和绝对路径
- **offset**：可选，读取起始位置（行号），默认为 1
- **limit**：可选，读取的最大行数，默认为 2000

## 使用场景

- 查看代码文件内容了解实现细节
- 读取配置文件了解应用设置
- 检查文档内容
- 分析项目结构
- 理解代码依赖关系

## 使用示例

```json
{
  "filePath": "src/main.rs",
  "offset": 1,
  "limit": 100
}
```

## 注意事项

1. **路径格式**：支持相对路径和绝对路径
2. **大文件处理**：读取大文件时建议使用 offset 和 limit 参数分段读取
3. **二进制文件**：此工具主要用于读取文本文件
4. **编码格式**：默认按 UTF-8 编码读取

## 读取策略

- 小文件：直接完整读取
- 大文件：使用 offset 和 limit 分段读取
- 定位内容：结合 grep 找到目标行后再读取
- 代码分析：从入口文件开始逐步深入

## 常见用途

- 理解项目结构时列出目录后读取关键文件
- 修复 bug 时读取相关代码
- 编写新代码时参考现有代码风格
- 审查代码时完整或部分读取文件
