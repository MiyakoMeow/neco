# 激活工具提示词

## 功能说明

`activate` 工具用于动态激活额外的技能模块或外部工具服务，增强 AI 的专业能力。

## 参数说明

- **name**：必填，要激活的技能或 MCP 服务器名称
- **type**：可选，激活类型（skill 或 mcp），默认为自动检测
- **command**：可选，MCP 服务器的启动命令
- **args**：可选，MCP 服务器启动参数列表

## 激活类型

### 1. 激活 Skills

Skills 是 NeoCo 提供的内置技能模块，可以增强 AI 在特定领域的能力。

#### Skills 分类

- **Rust 元认知技能**：处理 Rust 相关的编译错误、所有权问题、生命周期等
- **领域技能**：针对特定领域的专业能力，如 Web 开发、CLI、嵌入式等
- **工具技能**：提供特定工具的使用能力，如代码导航、依赖可视化等
- **最佳实践技能**：代码审查、反模式检测等

#### 使用示例

```json
{
  "name": "m01-ownership",
  "type": "skill"
}
```

#### 常用 Skills

- `rust-router` - Rust 综合问题处理
- `m01-ownership` - 所有权和借用问题
- `m03-mutability` - 可变性相关问题
- `coding-guidelines` - 代码规范指导

### 2. 激活 MCP 服务器

MCP（Model Context Protocol）是一种开放协议，允许 AI 助手连接外部工具和服务。

#### MCP 功能

通过 MCP，你可以：

- 访问外部 API 和服务
- 使用专门的开发工具
- 连接数据库和存储系统
- 集成第三方平台

#### 使用示例

```json
{
  "name": "filesystem",
  "type": "mcp",
  "command": "npx",
  "args": ["-y", "@modelcontextprotocol/server-filesystem", "/path/to/dir"]
}
```

## 使用场景

- 处理特定编程语言的编译错误时
- 需要使用特定领域工具时
- 需要使用外部 API 获取信息时
- 需要连接数据库进行查询时
- 需要专业领域的最佳实践指导时
- 进行代码审查或重构时

## 注意事项

1. **按需激活**：根据任务需求选择合适的技能或 MCP 服务
2. **技能组合**：可以同时激活多个相关技能
3. **资源消耗**：激活技能会占用额外资源，任务完成后可考虑停用
4. **配置要求**：MCP 服务器需要在配置中正确设置
5. **认证信息**：某些 MCP 服务需要提供 API 密钥或认证信息
